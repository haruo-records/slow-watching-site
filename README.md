# slow-watching-site

A definition page for **Slow Watching** — an attitude of perception — part of the HARUO concept network.

This is a static site built for GitHub Pages.

## About

Slow Watching names the practice of staying with what you observe before reaching for meaning. This site defines the concept clearly and quietly, and connects it to related ideas within the HARUO framework.

## File structure

```
slow-watching-site/
├── index.html   # Main page
├── styles.css   # All styles
└── README.md    # This file
```

## Deployment

This site is designed to run directly on GitHub Pages with no build step. Push to the repository and enable GitHub Pages from the repository settings.
